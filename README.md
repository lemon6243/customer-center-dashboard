# 🏢 도시가스 고객센터 성과 대시보드

24개 도시가스 고객센터의 월별 성과를 실시간으로 분석하고 시각화하는 Streamlit 기반 대시보드입니다.

## ✨ 주요 기능

- 📊 **전체 현황**: 센터별 순위 및 KPI 카드
- 📈 **월별 추이**: 누적 성과 그래프
- 🎯 **센터별 상세**: 레이더 차트 및 세부 점수
- ⚠️ **위험 관리**: 911점 미달 센터 자동 추출

## 🚀 빠른 시작

### 1. 로컬 실행

```bash
# 패키지 설치
pip install -r requirements.txt

# 앱 실행
streamlit run app.py
